使用（搜索力，提问力，批判力，结构力，创新力）

提示词框架（问法纲要）

  • 背景/规定情境：上下文信息
  • 角色详细设定：
    • 你希望ChatGPT扮演什么样的角色**
    • 「角色详细设定」-「行动最高目标/行动任务」·「表演要求」·「规定情境」**
    • GPT可以扮演任何人
      1. 如果你是xxx,在某种情况下，xxx会说什么，会怎么想，会怎么看，会怎么做？
      2. 直接扮演几个具体的人
      3. 扮演某几类人，可让他扮演多人进行讨论
  • 行动任务：具体执行什么任务
  • 输出关键结果：也就是输出的具体内容是什么以及格式
  • 辅助选项:其他个人要求，如结构化信息，限制信息，

  • Tips：

    • 用分隔符，比如```，”””,<>,,:
      • 使用了这些定界符，以使模型非常清楚地了解它应该总结哪些确切的文本
      • 具体一点，我在跟GPT debug的时候会在代码的前面和末尾都加上```
      • Summarize the text delimited by triple backticks into a single sentence. ‘’’{text}’’’.
    • 思维链(Chain of Thought)
      • 给予GPT按步思考的能力
      • 可以在输入辅助中添加这句话：Let’s work this out in a step by step way to be sure we have the right answer.
      • 向他展示一个类似问题的推理过程，让他学习
    • 提出子问题
      • 有些时候，中间的推理过程更为复杂一些，需要将问题分解成子问题；拆成子问题可以提高正确率
      • 也可一次性，将多个问题一起问
      • 给出多种方案（Self Consistency)
      • 给出一个问题的多个解决方案，最后综合考虑几个方案，得出最终的答案。
    • 让GPT帮你提问，或者问你

让GPT优化提示词：

  • 现在你需要充当 GPT 提示词优化器。我将提供一个提示，然后你将根据以下原则优化提示，使结果有效且相关。 原则如下：
    1. 明确定义提示的任务或作用；
    2. 提供清晰简洁的说明；
    3. 建立包含最佳实践和行业标准的评估标准；
    4. 为目标受众量身定制提示，提供相关资源；
    5. 参考示例优化的格式，但不要照抄。
    6. 只给我生成优化后的提示，不用解释，不用回答，不用翻译。
    • 示例： 优化前的提示：“有哪些B2B创业想法可以使用ChatGPT？” 经过你优化后的提示：“扮演一位连续创业者的角色。我会提供一些背景信息。B2B是指一个企业与另一个企业进行商业交易的情况。您将基于我的背景提供3个使用ChatGPT的B2B创业想法，它结合了两个极端情况：它应该冒很大的风险，但也应该容易实施。每个人都应该谈论它，但现有客户不会被打扰。结果应具体和建设性，并包括我需要调查下一步骤的详细信息。”
    • 如果你理解了，回复“OK”以确认，我将提供第一个提示。

提问纲要

提问前提（复杂问题：why,how)

  • why：GPT基于事实进行推理
  • how：将问题逐步清晰化，带有具体目标目的，再问实现目标的路径，最后问具体的方法
  • 关于交流：

元问题

  • 象限一：
    1. 我想了解XXX，我应该向你了解哪些问题
    2. 请给我列出XX领域/行业相关的最常用的50个概念，并做简单解释，如果有英文缩写请给出完整的英文解释
    3. 请详细介绍XX的生平事迹/请详细介绍XX公司的发展历程
  • 象限二：
    • 检验知识
      1. 对于XX主题/技能，你认为那些是我需要理解掌握的核心要点
      2. 我理解XX是这样的，你认为我的理解对吗
      3. 我对XX有一些想法，你能帮我批判的分析这些想法的优点和缺点嘛
      4. 我正在考虑XX的决定，你能帮我分析一下可能的结果和影响吗
    • 扩容知识
      1. 我知道XXX的概念，你能给我提出更多关于XXX的信息吗
      2. 我在XX问题上遇到了困难，你可以提供可能的解决方案和建议吗
      3. 我想深入学习XXX，你能推荐一些学习资源或者学习路径吗
      4. 我想在XXX领域有些创新，你能提供一些启发和想法吗
      5. 我想在某些领域有些提升，你能根据最新的研究和趋势给我一些建议吗
      6. 我正在考虑学习XXX，你能给我一些这个领域未来发展的观点吗
      7. （交代背景）我要做XXX的研究，我认为可能的原因是……还有其他可能的原因吗，给出一些可能的研究和假设
      8. 我是一个XX行业新手，马上要采访这个行业资深大佬，我应该向他请教哪些有价值的问题？
  • 象限三（前提：介绍背景，提供信息）：
    1. 你怎么看待这种现象
    2. 可能的原因有哪些
    3. 这会对XXX产生什么影响
    4. 你觉得XXX应该怎么做
  • 象限四（探索！！！）
    1. 如果XX发生，这对社会会产生什么影响

重点!!!

  • 达克效应

  • 反向提问:

    1. 为了测试我对xxx的了解程度，你会问我什么问题来检验我的水平，最少10个。
    2. 经典场景：请它以公司面试者身份对自己进行模拟面试
    3. 我是xx领域的专家,你会问我哪些问题来检验我的专业水平？最少10个。
    4. 这些我都知道，换一波更有深度、更有细节的
    5. GPT4专用:现在我们玩个“你问我答”的游戏,目的是为了测试我在xxxx方面的专业水平。你负责提问，我负责回答，你要根据我的答案进行反馈、评价、补充。如果我说不知道，你就直接输出正确答案，然后你继续提问。

批判性能力

  • 拆解分析，辅助深入理解： 我给你一段文字，你帮我详细描述以下的：观点，论题，证据事实、结论、推理解释的过程、假设、概念、立场分别是什么。文字如下：
  • 批判：请基于批判性思维，分别针对这个观点的：论题，证据事实、结论、推理解释的过程、假设、立场这7个论证部分进行批判。
  • 找错：结合各种论证谬误发问

结构化能力（externalization）

  • 要点：自己将隐性化的知识总结或让GPT总结，并让GPT基于这些总结的知识完成任务。

基于MECE分类方法的基本模型（Mutually Exclusive相互独立不重复Collectively Exhaustive完全穷尽不遗漏）

  • 二分法：如，主观/客观，输入/输出
  • 要素法：如，MECE分类方法，战略5c
  • 矩阵法：如，Johari沟通视窗
  • 过程法：如，流程图
  • 公式法：如，人生·成功=思维方式x热情x能力

其他结构化模型

  • 让gpt推荐不同结构的模型，如：请给我推荐20个最常用的商业分析模型
  • 让ChatGPT介绍模型用法、局限、举例
  • 让ChatGPT基于特定模型分析问题（特定场景&结构）
  • 让ChatGPT推荐特定行业/领域/人群的模型

创新力

  • 两个概念：
    1. 颠覆式创新：如iPhone4
    2. 延续性创新：小创新
  • 创新者的5工具：
    1. 观察
    2. 提问：结合结构，创新目标使用发问方法
    3. 联想：相关性
    4. 交流：建立多人场景头脑风暴
    5. 实验
