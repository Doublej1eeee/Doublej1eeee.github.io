登录

  • 可选择登录设置头像，昵称
  • 在登陆后可以购买vip以实现多台设备uTools插件的同步
  • 可不登录，在一台电脑内正常使用

使用

  • 快捷键：
    • alt+space：召唤uTools
    • ctrl+D：分离窗口快捷键
    • 超级面板：长按右键
    • 召唤翻译界面：双击ctrl
  • 召唤应用：直接输入应用名称
  • 启动系统级别的命令：
    1. 卸载软件：直接输入卸载
    2. windows cmd：直接输入打开
    3. windows power shell：直接输入打开
  • 快速搜索文件：输入find，再输入文件名
  • 备忘快贴：
    1. 直接输入建立备忘
    2. 搜索备忘进行查找
  • 翻译单词以及外文的段落：
    1. 方法一：针对选中的语段，单词在超级面板中翻译。左击鼠标键拉取选中，在长按右键即可。但每天有次数限制
    2. 方法二：在方法一的操作下，再超级面版中再点翻译即可。无使用次数限制。
    3. 方法三：双击ctrl，召唤翻译界面
  • ocr 识别：
    使用超级面版，选中截图并自动识别提取
  • 网页快开功能：
    1. 输入网页快开设置，在里面了解快捷键，或者自定义
    2. 输入应用名称，再在该应用中搜索，如：百度，哔哩哔哩，google
  • 插件市场：根据自己的需求使用
